# API For Godkoz Money Book

**_ใช้ yarn เท่านั้น_**

```js
npm install -g yarn
```

## Available Scripts

### `yarn start`

Runs the app in the development mode.\
on port `4000`

### `yarn build`

Builds the app for production to the `build` folder

### `yarn watch`

รันไฟล์ใน folder `/build/`

### `yarn clean`

ลบ folder `/build/`
